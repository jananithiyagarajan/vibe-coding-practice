# Module 9: AI Model Integration

This module teaches you how to work with AI models via APIs, focusing on effective prompting, response handling, and cost-aware development.

## Practicals

- `module-9-1-api-models-basics.md` — How API-based models work (Groq/OpenAI)
- `module-9-2-system-vs-user-prompt.md` — System prompt vs User prompt (Persona setting)
- `module-9-3-handling-responses.md` — Handling AI responses (JSON vs Text)
- `module-9-4-streaming-concepts.md` — Streaming concepts (Real-time AI)
- `module-9-5-cost-and-tokens.md` — Cost awareness and Token management
